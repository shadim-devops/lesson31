Zadanie 1: IAM Security

Cel

Konfiguracja bezpiecznego dostępu do zasobów AWS przy użyciu IAM.

Co zostało zrobione

* utworzono grupy użytkowników (Dev, QA, Ops)
* utworzono użytkowników i przypisano ich do grup
* włączono MFA dla użytkowników
* przypisano polityki dostępu (policies)
* utworzono rolę IAM
* skonfigurowano trust relationship

---

Zadanie 2: Monitoring Dashboard

Cel

Skonfigurowanie monitoringu zasobów AWS przy użyciu CloudWatch.

Co zostało zrobione

* utworzono instancję EC2
* skonfigurowano dashboard (CPU, Network)
* utworzono alarm dla CPU
* skonfigurowano powiadomienia SNS (email)
* utworzono log group oraz log stream
* wykonano zapytanie w Log Insights
